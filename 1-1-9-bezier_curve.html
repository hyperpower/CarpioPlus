2.1.4. Bezier Curve¶

2.1.4.1. Introduction¶

A Bézier curve is is a parametric curve used in computer graphics and related fields. Details can be found at Wiki. A few Bézier curves have been implemented.

2.1.4.2. Linear Bézier curves¶

Given distinct points \(\mathbf{P}_0\) and \(\mathbf{P}_1\), a linear Bézier curve is simply a straight line between those two points. The curve is given by

(2.1)¶\[\begin{split}\mathbf{B}(t) &= \mathbf{P}_0 + t(\mathbf{P}_1 - \mathbf{P}_0)\\ &= (1-t)\mathbf{P}_0 + t\mathbf{P}_1\end{split}\]

where, \(0 \le t \le 0\)

Fig. 2.17 An example of linear bezier curves.¶

2.1.4.3. Quadratic Bézier curves¶

A quadratic Bézier curve is the path traced by the function \(\mathbf{B}(t)\), given points \(\mathbf{P}_0\), \(\mathbf{P}_1\), and \(\mathbf{P}_2\),

(2.2)¶\[\mathbf{B}(t) = (1-t)[(1-t)\mathbf{P}_0 + t\mathbf{P}_1] + t[(1-t)\mathbf{P}_1 + t\mathbf{P}_2]\]

where, \(0 \le t \le 0\)

Fig. 2.18 An example of Quadratic bezier curves.¶

2.1.4.4. Cubic Bézier curves¶

Cubic Bézier curve can be defined as an affine combination of two quadratic Bézier curves:

Given points \(\mathbf{P}_0\), \(\mathbf{P}_1\), \(\mathbf{P}_2\) and \(\mathbf{P}_3\)

(2.3)¶\[\mathbf{B}(t) = (1-t) \mathbf{B}_{\mathbf{P}_0,\mathbf{P}_1,\mathbf{P}_2} + t \mathbf{B}_{\mathbf{P}_1,\mathbf{P}_2,\mathbf{P}_3}\]

where, \(0 \le t \le 0\)

Fig. 2.19 An example of Cubic bezier curves.¶

2.1.4.5. High Order Bézier curves¶

A recursive definition for the Bézier curve of degree n expresses it as a point-to-point linear combination (linear interpolation) of a pair of corresponding points in two Bézier curves of degree n − 1.

Let \(\mathbf {B}_{\mathbf {P} _{0}\mathbf {P} _{1}\ldots \mathbf {P} _{n}}\) denote the Bézier curve determined by any selection of points \(\mathbf{P}_0\), \(\mathbf{P}_1\), \(\ldots\) \(\mathbf{P}_n\). Then to start,

(2.4)¶\[\mathbf{B}(t) = (1-t) \mathbf{B}_{\mathbf {P} _{0}\mathbf {P} _{1}\ldots \mathbf {P} _{n-1}} + t \mathbf{B}_{\mathbf {P} _{1}\mathbf {P} _{2}\ldots \mathbf {P} _{n}}\]
