2.2.2. Jacobi Solver¶

2.2.2.1. Problem Description¶

This example shows Jacobi method for solving \(\mathbf{A}x = \mathbf{b}\).

2.2.2.2. Test Problem¶

The Matrix is obtained from MatrixMarket. The name is 685_bus.mtx. The equation which we want to solve:

\[\mathbf{A} \cdot \mathbf{x} = \mathbf{b}\]

where, \(\mathbf{A}\) the matrix, \(\mathbf{x}\) unknown vector and \(\mathbf{b}\) is the vector on right hand side.
